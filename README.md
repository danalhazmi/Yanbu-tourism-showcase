# Yanbu Industrial City - Tourism & Showcase Portal

A responsive, semantic multi-page web platform dedicated to showcasing the rich heritage, advanced infrastructure, and tourism landmarks of Yanbu Industrial City. Developed using modern frontend standards as part of the Advanced Topics in E-Commerce course.

## 🚀 Live Demo
🌐 **https://danalhazmi.github.io/Yanbu-tourism-showcase/**

## ✨ Key Features
- **Semantic Architecture:** Built entirely using structural HTML5 elements (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>`) for maximum SEO readiness and accessibility.
- **Modern Responsive Design:** Custom-styled component cards styled with precise CSS padding, margins, and smooth transition hover effects.
- **Advanced Visual Presentation:** Employs dynamic box-shadow properties and responsive layouts to ensure a clean user interface across desktop and mobile devices.
- **Native Multimedia Integration:** Features fully controlled interactive elements including a modern landscape video showcase and automated legacy audio streams capturing local maritime music.
- **Multipage Navigation:** Fully structured directory covering Home, About the City, Places to Visit, Food & Culture, and Contact interfaces.

## 🛠️ Tech Stack & Tools
- **Frontend:** HTML5, CSS3, JavaScript
- **Multimedia:** Native HTML5 Video & Audio API
- **Design Philosophy:** "Old Money" Inspired Aesthetic / Professional Business Layout
- **Environment:** Visual Studio Code

## 📁 Repository Structure
- `index.html` - Main portal landing page
- `about.html` - Historical and infrastructure context
- `places.html` - Tourism and landmarks directory
- `food.html` - Local cuisine and traditional culture showcase
- `contact.html` - User inquiry form
- `style.css` - Centralized cascading stylesheet containing responsive rules
- `/media` - Directory containing verified high-resolution graphic and native media assets
